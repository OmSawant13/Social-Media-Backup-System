## 🚀 Quick Start

### 1. Start MongoDB
```bash
mongod
```

### 2. Start Backend Server
```bash
cd backend
npm install
node server.js
```

### 3. Start Frontend Server
```bash
cd frontend
python3 -m http.server 8002
```

### 4. Access Application
- **Landing Page:** http://localhost:8002/landing.html
- **Main App:** http://localhost:8002/index.html
